Student Management System

-Overview
The Student Management System is an ongoing project designed to streamline the management of student information and academic records. This system aims to provide an efficient way for educational institutions to manage student data, including enrollment, attendance, grades, and more.

-Features

User Authentication: Secure login for administrators and users.
Student Enrollment: Add, update, and delete student records.
Course Management: Manage courses and assign students to courses.
Attendance Tracking: Record and monitor student attendance.
Grade Management: Input and manage student grades.

-Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: Laravel
Database: MySQL
